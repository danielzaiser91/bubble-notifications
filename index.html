How it works

  1. Import the BN class and its stylesheet, then call BN.init() once the DOM is ready. The container is created automatically and pinned to the bottom center of the viewport — or add your own <div id="infoBubbleContainer"> if you want to control where it lives. This page uses the automatic one.
  2. Call BN.info("your message") whenever you want a bubble. It pops in, floats for 2.5 seconds and removes itself — multiple bubbles stack automatically. Pass a second argument to change the duration: BN.info("Slow…", 6000).
  3. Restyle the bubbles with plain CSS custom properties — colors, padding, radius (see below).

Installation & usage

1 · Install from npm

npm install @danielzaiser91/bubble-notifications

2 · Add the stylesheet

<!-- in your HTML, or import the .css via your bundler -->
<link rel="stylesheet" href="node_modules/@danielzaiser91/bubble-notifications/bubble-notifications.css">

3a · Use it (ES modules)

import { BN } from "@danielzaiser91/bubble-notifications";

BN.init();                             // once, after the DOM is ready
BN.info("Hello from a bubble!");      // anywhere, as often as you like
BN.info("I stay longer…", 6000);      // optional: duration in ms (default 2500)

3b · Use it (CommonJS)

const BN = require("@danielzaiser91/bubble-notifications");

BN.init();
BN.info("Hello from a bubble!");

3c · No build step? Use a CDN

<link rel="stylesheet" href="https://unpkg.com/@danielzaiser91/bubble-notifications/bubble-notifications.css">

<script type="module">
  import { BN } from "https://unpkg.com/@danielzaiser91/bubble-notifications/bubble-notifications.js";
  BN.init();
  BN.info("Ready to go!");
</script>

Customizing

The bubbles read their look from CSS custom properties — set them on the container (this is exactly how the theme switcher above works):

#infoBubbleContainer {
  --bn-background: #23283f;   /* any background, gradients included */
  --bn-color: #ffd400;        /* text color */
  --bn-padding: 12px 20px;
  --bn-radius: 8px;
}

For anything beyond that, just override .infoBubble or #infoBubbleContainer (position) directly.
